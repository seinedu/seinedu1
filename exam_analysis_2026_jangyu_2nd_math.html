서10 분수형 부등식 상수 역산 (9점 — 최고 배점)
(-3x+1)/2 + 2 ≤ -(a+2x)/3, 해가 x≥2 → 풀면 x≥(2a+15)/(-5) 형태 → 경계값 대입해 a 역산. 부등호 방향 전환 + 분수 통분 + 상수 역산 3단 복합. 계산 오류 없이 서술 완성 필수.
상 9점 직결
02
서6 나눗셈 오류 역산 + 서7 y 대입 (각 8점)
서6: -½x²y로 나눠야 할 것을 곱함 → 결과 -16x²+4y에서 원래 식 역산 → 올바른 계산(-4x⁶y²+x⁴y³). 서7: y=-x+2 대입 후 2x(x-y+1)-(3x²y+xy²-4xy)÷y/3 정리 → -2x²+4x.
상 16점 직결
03
서9 복합 조건 교집합 (8점)
조건 3가지: ① 한 자리 자연수 ② 14/(50×x) 유한소수 → 분모 소인수 2·5만 → x 후보 추출 ③ 3(x-3)>6-2x → x>3. 세 조건 교집합으로 x 확정 → 합 계산. 순서 바꾸면 오류.
상 8점 직결
04
선4 마방진 + 선5 계수 조건 (각 6점)
선4: 가로·세로·대각선 합이 같도록 ★ 식 결정 — 지수법칙 응용 + 등식 설정. 선5: x²계수=x계수 조건 → a 값 계산. 두 문항 모두 선택형 최고 배점으로 풀이 과정 없이 정답만 요구.
