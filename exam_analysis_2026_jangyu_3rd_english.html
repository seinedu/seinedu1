출제 문법 4종
① 관계대명사 / 관계부사
선3·선10·서6·서12(A) — 관계부사(the way how → 불가), 관계대명사 what(선행사 없음), 관계대명사 who/which/that로 두 문장 연결. 서6 두 문장 연결 영작(8점)이 핵심.
28점 이상 연관
② 현재완료 완료형 (have+p.p.)
서12(D)(F) — D: has been knowing(상태동사 진행 불가) → has known. F: has been not doing → has not been doing(부정어 위치). 현재완료 진행과 단순 완료 구분이 핵심.
8점 직결 (서12)
③ I-message / 의사소통 기법
서5(5점)·서7(4점) — "You are so selfish!" → I-message로 전환. "Don't put your bag" → Could you please 요청형으로 변환. 교과서 핵심 표현 + 조건 영작 결합.

04 / STRATEGY
세인학원 학습 전략
우선순위별 기말고사 집중 전략
01
서12 어법 오류 3문장 교정 (12점 — 단일 최고 배점)
A: the way how → the way 또는 how (관계부사 중복 불가). D: has been knowing → has known (know는 상태동사, 진행형 불가). F: has been not doing → has not been doing (부정어는 have 뒤). 오류 식별 + 문장 전체 재작성까지 완벽 서술 필수.
상 12점 직결
02
서6 관계대명사/관계부사 두 문장 연결 (8점)
(1) Maybe Calli's mom wanted to listen to the thing. / Calli was going to say it. → ...the thing that Calli was going to say. (2) Calli should pick a time. / Her mom is ready to listen at the time. → ...a time when her mom is ready to listen. 관계대명사 that·관계부사 when 선택 + 연결 완성.
상 8점 직결
03
서8 어법 오류 2문장 교정 + 밑줄 재작성 (8점)
석조 숲 지문 ⓐ~ⓔ 중 틀린 두 개 찾아 밑줄 친 부분 교정. ⓓ: The environment is harsh of animals → for animals to live in / ⓔ: that only live → which only live (선행사가 lemurs). 각각 4점, 교정 부분만 고치면 됨.
상 8점 직결
04
서2 조건 영작 + 서5 I-message 영작
서2(6점): (가) that's not what I meant 활용 + (나) took/way 활용 서로 다른 표현. 서5(5점): "You are so selfish!" → I feel unhappy because I have to wait longer for my meal. 조건 단어 모두 사용.
상 11점 직결
05
선택형 10문항 + 독해 기반 서술형 기본 점수 사수
선택형 10문항(28점) — 대화순서·불일치·어휘 기본. 서3 관용구(4점)·서4 영어 답변(4점)·서10 인터뷰 빈칸(8점) — 독해 기반 16점은 안정 확보 가능.
